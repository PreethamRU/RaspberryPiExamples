# RaspberryPiExamples
This repository consists of RaspberryPI with Python and Ubuntu. 

you can refer to the various examples. 

1) Clean the SD Card
2) Upload OS to RPi
3) Install Python (lates version of Ubuntu comes with Python)
4) Start running these programs by connecting to respective sensors to pins.

note: Example I've taken here is AWS Cloud IoT. You can refer to your own cloud.
